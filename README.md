# Sprint_6
# Автотесты для сервиса заказа самокатов

##  Описание проекта

Проект содержит автоматизированные тесты для проверки функционала сервиса заказа самокатов:
- Проверка работы выпадающего списка FAQ
- Позитивные сценарии оформления заказа через верхнюю/нижнюю кнопку
- Проверка редиректов по логотипам
- Проверка доступности элементов интерфейса

##  Тестируемая функциональность

### 1. Тестирование заказа самоката
- Тест 1: Заказ через верхнюю кнопку "Заказать" (параметризован - 2 набора данных)
    Проверка оформления заказа с минимальными данными
    Проверка оформления заказа с альтернативными данными
- Тест 2: Заказ через нижнюю кнопку "Заказать" (параметризован - 1 набор данных)
    Проверка оформления заказа с полными данными
- Тест 3: Проверка редиректа по логотипу Самоката
    После перехода на страницу заказа, клик на логотип возвращает на главную
- Тест 4: Проверка редиректа по логотипу Яндекса
    Клик на логотип открывает новую вкладку с Яндекс.Дзен
- Тест 5: Проверка видимости верхней кнопки "Заказать"
    Элемент присутствует и отображается на главной странице
- Тест 6: Проверка видимости нижней кнопки "Заказать"
    Элемент присутствует и отображается на главной странице

### 2. Тестирование FAQ раздела
- Проверка выпадающего списка "Вопросы о важном" (параметризован - 8 вопросов)
