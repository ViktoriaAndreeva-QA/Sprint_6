# Sprint_6
# Автотесты для сервиса заказа самокатов

##  Описание проекта

Проект содержит автоматизированные тесты для проверки функционала сервиса заказа самокатов:
- Проверка работы выпадающего списка FAQ
- Позитивные сценарии оформления заказа
- Проверка редиректов по логотипам
- Проверка доступности элементов интерфейса

##  Структура проекта

Sprint_6/
├── tests/                                      # Тестовые файлы
│   ├── test_order_scooter_positive_flow.py     # Тесты заказа самоката
│   └── test_faq_section.py                     # Тесты FAQ
│
├── pages/                                      # Page Object классы

│   └── order_page.py                           # Страница заказа
│

│   ├── main_page_locators.py
│   └── order_page_locators.py

##  Тестируемая функциональность

### 1. Тестирование заказа самоката
- **Тест 1**: Заказ через верхнюю кнопку
- **Тест 2**: Заказ через нижнюю кнопку
- **Тест 3**: Заказ через верхнюю кнопку (альтернативные данные)
- **Тест 4**: Проверка редиректов по логотипам
- **Тест 5**: Проверка обеих кнопок заказа

### 2. Тестирование FAQ раздела
- Проверка всех 8 вопросов в выпадающем списке
- Каждый вопрос тестируется отдельно
- Проверка отображения соответствующего текста
